# Technical Status

מצב טכני נוכחי של הפרויקט.

## Main branch

ענף `main` מיועד לשינויים יציבים בלבד:

- תוכן האתר
- עיצוב ו-UX
- קבצי SEO/GEO
- דוקומנטציה
- תיקוני באגים קטנים

## Analytics

כרגע ב-main פעיל רק Vercel Web Analytics.

PostHog אינו חלק מענף `main` כרגע.

אם מוסיפים PostHog בעתיד, זה צריך להיות סבב נפרד עם מדידה מינימלית בלבד.

אירועים אפשריים:

- פתיחת מקור
- העתקת תגובה קצרה
- מעבר לעמוד בדיקה
- פתיחת FAQ

מה לא למדוד:

- תוכן חופשי של משתמשים
- הקלטות מסך
- פרופיילינג אישי
- URLים רגישים

## Video

Remotion אינו חלק מענף `main` כרגע.

אם מוסיפים Remotion בעתיד, המטרה צריכה להיות ברורה:

- יצירת סרטוני הסבר קצרים מתוך עמודי בדיקה
- יצירת קליפים לרשתות
- יצירת ויזואליזציות מבוססות תוכן קיים

לא להוסיף Remotion רק כי הוא מעניין. הוא מוסיף תלות, תחזוקה וזמן build.

## Next.js upgrade

שדרוג Next.js נשאר לסוף.

כללי עבודה:

1. ליצור ענף ייעודי.
2. לא לערבב עם PostHog או Remotion.
3. להריץ `npm run typecheck`.
4. להריץ `npm run build`.
5. לבדוק עמוד בית, עמוד בדיקה, sitemap, robots ו-llms.txt.
6. רק אז למזג ל-main.
