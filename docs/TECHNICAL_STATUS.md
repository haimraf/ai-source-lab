# Technical Status

מצב טכני נוכחי של הפרויקט.

## Current stage

הפרויקט כבר לא רק MVP ראשוני. הוא מאגר בדיקות מקור פעיל ובטא ציבורית עם route דינמי ל-claims, תוכן typed, חיפוש מקומי, אשכולות נושא, OG, sitemap ו-llms.

היעד הטכני הקרוב הוא לא להוסיף פיצ׳רים כבדים, אלא לשמור על יציבות בזמן שהמאגר גדל מעבר ליעד התוכן הראשוני של 24 בדיקות ולכיוון 50-70 בדיקות. ספירת הבדיקות צריכה להיגזר מרשומות התוכן ומהאתר החי, לא מספירה ידנית במסמך הזה.

## Main branch

ענף `main` מיועד לשינויים יציבים בלבד:

- תוכן האתר
- שיוף בדיקות קיימות
- עיצוב ו-UX קטן
- קבצי SEO/GEO
- דוקומנטציה
- תיקוני באגים קטנים

## Content architecture

- תוכן הבדיקות נמצא תחת `content/claims/`.
- `content/claims/index.ts` הוא נקודת הרישום של כל הבדיקות.
- `lib/content/claim-loader.ts` מספק טעינה לפי slug, path ואשכול.
- `lib/claims-db.ts` הוא אינדקס נגזר לצרכנים קיימים כמו עמוד הבית, מפת הנושאים וה-featured claim.
- עמודי הבדיקות מרונדרים דרך `app/claims/[slug]`.
- תמונות OG נשארות לכל claim בתיקייה המתאימה תחת `app/claims/<slug>/opengraph-image.tsx`.

## Analytics

כרגע ב-main פעיל רק Vercel Web Analytics.

PostHog אינו חלק מענף `main` כרגע.

אם מוסיפים PostHog בעתיד, זה צריך להיות סבב נפרד עם מדידה מינימלית בלבד.

אירועים אפשריים:

- פתיחת מקור
- העתקת תגובה קצרה
- מעבר לעמוד בדיקה
- פתיחת FAQ

מה לא למדוד:

- תוכן חופשי של משתמשים
- הקלטות מסך
- פרופיילינג אישי
- URLים רגישים

## Video

כלי Remotion וקובצי הווידאו נמצאים בריפו לצורך יצירת נכסי הפצה, אך אינם נטענים באתר ואינם חלק מ-runtime של Next.js.

אם מוסיפים Remotion בעתיד, המטרה צריכה להיות ברורה:

- יצירת סרטוני הסבר קצרים מתוך עמודי בדיקה
- יצירת קליפים לרשתות
- יצירת ויזואליזציות מבוססות תוכן קיים

לא להוסיף Remotion רק כי הוא מעניין. הוא מוסיף תלות, תחזוקה וזמן build.

## Next.js upgrade

הגרסה הפעילה היא Next.js 16.2.9. יש לעקוב אחר `npm audit` ולעדכן בענף ייעודי כאשר מתפרסם תיקון אבטחה רלוונטי.

שדרוג Next.js נשאר לסוף.

כללי עבודה:

1. ליצור ענף ייעודי.
2. לא לערבב עם PostHog, Remotion או שיוף תוכן.
3. להריץ `npm run typecheck`.
4. להריץ `npm run build`.
5. לבדוק עמוד בית, עמוד בדיקה, sitemap, robots ו-llms.txt.
6. רק אז למזג ל-main.
